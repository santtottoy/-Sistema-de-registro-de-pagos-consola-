# Registro de Pagos - Consola C#

Aplicación de consola desarrollada en C# como práctica de lógica de programación.

## Descripción
El programa permite registrar pagos de clientes y determina si el pago es:
- Exacto
- Incompleto
- Con vuelto

Además, muestra un resumen final con el total de clientes y el dinero recibido.

## Tecnologías usadas
- C#
- .NET
- Aplicación de consola

## Conceptos aplicados
- Condicionales (if / else)
- Bucles (for)
- Contadores
- Acumuladores
